# Когортный анализ удержания клиентов

## Обзор проекта

В этом проекте проводится анализ удержания клиентов с использованием когортного анализа на основе бразильского набора данных электронной коммерции Olist.
Цель — понять, сколько клиентов возвращаются после первой покупки и как меняется уровень удержания с течением времени.


## Набор данных

* Источник: Открытый набор данных по электронной коммерции Бразилии от Olist
* Включает данные о заказах, клиентах и ​​платежах

---

## Используемые инструменты

* SQL (PostgreSQL)
* Excel (сводные таблицы, тепловая карта)

## Результаты

* Удержание клиентов крайне низкое
* Менее 1% пользователей возвращаются после первой покупки
* Большинство клиентов совершают только одну покупку


## Ключевые выводы

* У компании низкий уровень удержания клиентов
* Клиенты ведут себя как разовые покупатели
